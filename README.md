# Swift To Do App

## Overview

This is a beginner Swift to do list app built to practice basic app development, user interaction, and foundational programming logic.

The project helped me better understand how a mobile app is structured and how code can be used to create simple task management functionality.

## Purpose

I created this project as part of my technical preparation for graduate study in Information Systems. Coming from a film and media background, I wanted hands on experience building a functional app while strengthening my programming foundation.

## Skills Practiced

- Swift programming fundamentals
- App structure and organization
- User interaction
- Variables and functions
- Basic task management logic
- Debugging and problem solving
- Interface flow and usability

## What I Learned

Through this project, I practiced how to think through an app’s structure, how user actions connect to app logic, and how small programming decisions affect the overall user experience.

This project also helped me build confidence working with code and understanding how applications are developed from an idea into a working product.

## Project Status

Completed as a beginner learning project.
